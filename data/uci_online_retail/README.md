# Downloaded UCI Online Retail data

This directory keeps only instructions in Git. `scripts/import_uci.py` writes
the downloaded archive, extracted workbook, converted CSV, and provenance JSON
here; those generated files are ignored.

Source: Daqing Chen, *Online Retail*, UCI Machine Learning Repository, dataset
352, DOI <https://doi.org/10.24432/C5BW33>, licensed CC BY 4.0.
